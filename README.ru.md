# printbookpdf

**Форматировщик PDF в буклет для двусторонней печати**

</div>

Переставляет страницы PDF в порядок буклета (сигнатуры) для двусторонней печати. Принимает любой PDF, дополняет его до кратного 4 числа страниц, переупорядочивает страницы так, чтобы при двусторонней печати и сложении они образовывали правильно последовательный буклет. Страницы на обороте каждого листа повёрнуты на 180 градусов, чтобы обе стороны совпадали.

## ■ Возможности

- ❖ **Переупорядочивание в буклет** — соединяет последнюю страницу с первой, предпоследнюю со второй и так далее в порядок сигнатуры
- ❖ **Автодополнение** — добавляет пустые страницы формата Letter (612×792 pt) до кратного 4
- ❖ **Поворот страниц** — поворачивает две из каждых четырёх страниц на 180° так, чтобы оборотная сторона печаталась правильно
- ❖ **Любой входной PDF** — работает с документами любого количества страниц

## ■ Как это работает

```
1. Загрузить входной PDF с помощью PyPDF2.
2. Добавлять пустые страницы формата Letter (612×792 pt) до тех пор, пока количество страниц не станет кратным 4.
3. Переупорядочить страницы в порядок сигнатуры: соединить последнюю страницу с первой, предпоследнюю со второй и т.д.
4. Повернуть каждую вторую страницу в каждой паре на 180°, чтобы оборотная сторона печатного листа читалась правильно.
5. Записать результат в директорию books/ как <input-name>.pdf.
```

## ■ Использование

```bash
pip install "PyPDF2<3.0"
python printBookPDF.py input.pdf
```

> Примечание: выходная директория жёстко задана в `printBookPDF.py` (`DIR = '/Users/pluttan/Desktop/проекты/printbookpdf/books/'`). Отредактируйте эту строку, указав свою папку `books/`, перед запуском — результат сохраняется туда как `<input-name>.pdf`.
